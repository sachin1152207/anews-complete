
# Anew Nepal 

"A New Nepal" is a full-stack website that has been developed by Sachin Shrivastav. The website features user management, Facebook and YouTube API integration, comment management, and post management. The website aims to provide a platform where people can share their views and ideas about Nepal.

The user management feature allows users to create accounts and log in to the website. The Facebook and YouTube API integration allows users to easily share and embed videos and posts from those platforms onto the website. The comment management feature enables users to leave comments on posts and engage in discussions with other users. The post management feature allows users to create, edit, and delete their own posts on the website.

Overall, "A New Nepal" is a dynamic and interactive platform that allows users to express their opinions, share their experiences, and connect with others who have similar interests. It serves as a valuable resource for anyone looking to learn more about Nepal and engage with its people.
